##### 1.chrome快捷键

| 快捷键           | 功能                   |
| ---------------- | ---------------------- |
| ctrl + t         | 打开并跳转到新页面     |
| ctrl + w         | 关闭当前页面           |
| ctrl + Tab       | 切换到下一个页面       |
| ctrl + shift + t | 打开最近一次关闭的页面 |
| ctrl +  d        | 添加当前页面到书签     |
| ctrl + s         | 保存当前页面           |
| ctrl + r         | 刷新当前页面           |
| ctrl + h         | 打开历史记录页面       |
| ctrl + j         | 打开下载类容页面       |
| ctrl + shift + o | 打开书签管理器页面     |

##### 2.google search 技巧

1.完全匹配 `""`

```bash
"key"
```

2.排除 `-`

```bash
key -notKey #-没有空格
```

3.匹配 `*`

```bash
key*key
```

4.网址 `site`

```bash
key site:address #指定网址
key -site:address #排除网址
```

5.文件类型 `typefile`

```bash
key typefile:pdf/ppt/docx/doc/mobi
```
